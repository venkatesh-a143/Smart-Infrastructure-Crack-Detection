# Smart Infrastructure Crack Detection

## Project Overview
This project aims to develop a system for detecting cracks in various types of infrastructure, such as bridges, roads, and buildings. The system utilizes advanced image processing and machine learning techniques to identify and classify cracks, providing crucial data for maintenance and safety assessments.

## Setup Instructions
1. **Clone the Repository**: 
   ```bash
   git clone https://github.com/venkatesh-a143/Smart-Infrastructure-Crack-Detection.git
   ```
2. **Navigate into the Project Directory**:
   ```bash
   cd Smart-Infrastructure-Crack-Detection
   ```
3. **Install Dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
4. **Run the Application**:
   ```bash
   python app.py
   ```
5. **Access the Web Interface** at `http://localhost:5000` to use the crack detection system.
